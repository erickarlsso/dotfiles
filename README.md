# dotfiles

## Homebrew

### Install
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Update
```bash
brew update
```

## Install MacOS-applications
```bash
brew install --cask \
  keeper-password-manager \
  firefox \
  brave-browser \
  iterm2 \
  visual-studio-code \
  sublime-text \
  docker \
  jiggler \
  docker
```

## Install terminal-applications
```bash
brew install \
  wget \
  git \
  nvm
 ```
